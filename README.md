# BurgerArchitect

[![C#](https://img.shields.io/badge/C%23-.NET-512BD4?logo=dotnet&logoColor=white)](https://dotnet.microsoft.com/)

Projet applicatif .NET orienté architecture et modélisation métier.

## Fonctionnalités

- Modélisation d'un domaine applicatif clair
- Structuration des couches et responsabilités
- Base pédagogique ou professionnelle réutilisable

## Stack technique

- C#
- .NET
- Solution Visual Studio

## Prérequis

- .NET SDK
- Visual Studio ou terminal dotnet

## Démarrage rapide

```bash
dotnet restore
dotnet build
```

## Commandes utiles

```bash
dotnet build
dotnet test
```
